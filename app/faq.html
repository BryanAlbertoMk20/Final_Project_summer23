Frequently Asked Questions

What is the purpose of this site?
To have an integrated environment that allows for focused study sessions and calming music.
Can I add my own music to the site?
Currently there is no option to add or remove songs but you can mute the available track and play your own if you'd like.
What is the point of the timer in the notes section?
This is so that if anyone wants to time themselves and see how long it takes them to study a topic or if they would like to do study technques like the Pomodoro timing.
Why use this site?
The purpose of this site is to have the student ebe able to take notes listen to music and time themselves all in one place. The same task would need three different windows to be open to acomplish the same task.
